Se você fez o upload dos arquivos diretamente pela interface do site do GitHub (ou via comando push), o Streamlit Cloud deve, em teoria, atualizar sua aplicação automaticamente.

Aqui está como verificar se a atualização funcionou ou o que fazer se ela estiver "travada":

1. Verifique o "Deploy" Automático
Ao entrar na URL da sua aplicação no navegador:

Olhe para o canto superior direito. Se aparecer uma mensagem como "Running..." ou um ícone de carregamento, o Streamlit já percebeu a mudança no GitHub e está reinstalando os arquivos.

Pressione Ctrl + F5 (ou Cmd + Shift + R no Mac) para limpar o cache do navegador e garantir que você não está vendo uma versão antiga salva no seu computador.

2. O que fazer se não atualizar sozinho?
Às vezes, a conexão entre o GitHub e o Streamlit Cloud pode demorar alguns minutos. Se nada mudar, siga estes passos:

Acesse o seu Dashboard do Streamlit Cloud.
https://streamlit.io/cloud
Join Community Cloud 

Localize o seu app na lista.  => gutpassos' apps => chatfolha ∙ main ∙ app.py

Clique nos três pontinhos (...) ao lado do nome do app.

Selecione Reboot app. Isso forçará o servidor a baixar a versão mais recente do seu código no GitHub e reinstalar tudo.
