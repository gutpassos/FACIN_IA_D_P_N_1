# Documentação Gerada

Esta pasta contém documentação gerada automaticamente pelo sistema CI/CD.

## Conteúdo

- **BUILD_INFO.txt**: Informação de build e data de geração
- **API Documentation**: Documentação de APIs (quando disponível)
- **Code Coverage**: Relatório de cobertura de testes (HTML)

## Atualização Automática

Esta documentação é atualizada automaticamente:
- A cada push para `main` ou `develop`
- A cada merge de Pull Request aprovado
- Diariamente via cronograma (0:00 UTC)

Para visualizar a documentação mais recente, verifique a data em `BUILD_INFO.txt`.

---

**Nota**: Esta pasta é gerada automaticamente e não deve conter commits manuais.
